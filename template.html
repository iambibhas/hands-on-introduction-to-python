What is Python?

What is Python?

  • A dynamic, interpreted, high-level language
  • Does not require declaring the types of variables or parameters or methods
  • Extremely readable
  • Used for solving a *really* wide range of problems
  • Object-oriented
  • Does not force Object-orientation
  • Has and requires common sense

Data Structure: List

>>> mylist.append(1)
>>> mylist.append(2)
>>> mylist.append(3)
>>> print(mylist[0]) # prints 1
1
>>> print(mylist[1]) # prints 2

>>> print(mylist[2]) # prints 3

>>>

...     print x
...

Data Structure: Dictionary

>>> phonebook = {}
>>> phonebook["John"] = 938477566
>>> phonebook["Jack"] = 938377264
>>> phonebook["Jill"] = 947662781
>>> print phonebook
{'Jill': 947662781, 'John': 938477566, 'Jack': 938377264}
>>> print phonebook['Jill']
947662781
>>> print phonebook.get('Jack')
938377264
>>> print phonebook.get('Brian', 'No number found')
No number found
>>> 'Jill' in phonebook
True
>>> phonebook.keys()
['Jill', 'John', 'Jack']
>>> phonebook.values()
[947662781, 938477566, 938377264]
